# 🍢 Warung Om Uyung — Website Profile

> **Makan. Minum. Santai.** — Tempat nongkrong paling asik di Bondowoso, Jawa Timur.

---

## 📌 Deskripsi

Website profil resmi **Warung Om Uyung**, sebuah warung nongkrong lokal yang berlokasi di Bondowoso, Jawa Timur. Website ini menampilkan informasi lengkap tentang warung mulai dari sejarah berdirinya, menu makanan dan minuman, galeri suasana warung, hingga informasi kontak dan lokasi.

---

## 🌐 Demo

🔗 [http://warung-om-uyung.rpldev.org/](http://warung-om-uyung.rpldev.org/)

---

## 📋 Fitur Website

- **Hero Section** — Tampilan utama dengan tagline *Makan. Minum. Santai.*
- **Tentang Kami** — Pengenalan singkat tentang Warung Om Uyung
- **Sejarah** — Timeline perjalanan warung dari 2016 hingga sekarang
- **Menu** — Daftar menu lengkap dengan foto, deskripsi, dan harga (filter makanan/minuman)
- **Suasana (Vibes)** — Highlight fasilitas dan suasana warung
- **Galeri** — Foto momen pengunjung bersama di warung
- **Kontak** — Informasi kontak, jam buka, dan Google Maps embed
- **Dukungan Bahasa** — Tersedia opsi bahasa Indonesia & Inggris (EN)

## 👨‍🍳 Tentang Pemilik

Warung Om Uyung dirintis oleh **Om Uyung** bersama istrinya **Risnani** sejak 2016. Dari lapak kecil di pinggir jalan, kini telah berkembang menjadi warung nongkrong favorit dengan ratusan pelanggan setia di Bondowoso.
